# Querio

Querio is a multilingual university query resolution platform designed to bridge communication gaps between students and institutions. It enables users to ask questions in multiple languages and receive simulated responses instantly, while also providing centralized notices, FAQs, and feedback features.

---

## Features

- Simulated Authentication (Login / Registration)
- Multilingual UI (English, Hindi, Tamil, Telugu)
- Simulated AI Chatbot with language detection
- Chat History (Session-based)
- Notice & Announcement Board (Expandable cards)
- FAQ Section (Accordion layout)
- Feedback & Rating System
- Responsive, corporate UI (Navy + White theme)

---

## Project Structure

The application follows a Single Page Application (SPA) architecture with dynamic routing and state-based UI rendering.

### Main Sections

- Landing Page
- Authentication (Login / Register)
- Dashboard
- Chat Interface
- Chat History
- Notices
- FAQ
- Feedback

---

## Language Support

The platform includes a UI language selector that dynamically translates:

- Navigation
- Notices
- FAQ content
- Chatbot default responses

Translations are currently hardcoded for simulation purposes.

---

## Tech Overview

- Frontend-only implementation
- Local state simulation (no backend)
- Session-based data persistence
- Clean, grid-based layout
- Corporate minimal design (Navy + White)

---

## Purpose

Querio was developed as part of a Software Engineering and Project Management project to demonstrate:

- User story implementation
- Agile backlog mapping
- UI flow clarity
- Multilingual system design
- Low-fidelity to high-fidelity prototyping workflow
